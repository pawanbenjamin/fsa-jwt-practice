🔐 JWT Authorization Endpoint

This API contains two endpoints. One for ✍️'signing up' and one for 🤝'authenticating' a user.

  • /signup

    Must pass an object with username and password properties in request body.

    Returns an object with success, message and token properties

  • /authenticate

    The incoming request must send the token back in authorization header in following format:

    Authorization: `Bearer ${token}`

    On success, will return an object with success, message and data properties
